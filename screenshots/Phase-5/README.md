# Phase 5 – Microsoft Defender Administration

## Objective

Review and verify Microsoft Defender Antivirus settings to ensure the workstation is protected before deployment.

---

## Tasks Completed

- Reviewed Microsoft Defender Antivirus
- Verified protection settings
- Reviewed scan options
- Checked protection history
- Reviewed exclusions
- Verified protection updates
- Confirmed Windows Security providers

---

## Skills Learned

- Microsoft Defender Administration
- Antivirus Configuration
- Malware Protection
- Security Monitoring
- Windows Endpoint Protection

---

## Enterprise Best Practices

- Keep Real-time Protection enabled.
- Install Security Intelligence updates regularly.
- Avoid unnecessary exclusions.
- Monitor Protection History.
- Verify all Windows Security components before deployment.

---

## Outcome

Microsoft Defender has been reviewed and verified. The workstation meets the organization's endpoint protection requirements.
