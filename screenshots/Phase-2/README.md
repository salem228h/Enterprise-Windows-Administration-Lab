# Phase 2 – Workstation Standardization

## Objective
Prepare a newly installed Windows 11 workstation according to company standards before assigning it to an employee.

## Tasks Completed

- Renamed the workstation
- Installed Windows Updates
- Verified system activation
- Verified date, time, and time zone
- Reviewed Remote Desktop configuration
- Reviewed Power Plan and Sleep settings
