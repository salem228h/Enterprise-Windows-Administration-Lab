# Phase 1 – Windows Baseline Documentation

## Objective

The objective of this phase is to collect baseline information from a newly installed Windows 11 Enterprise workstation before making any configuration changes.

Creating a baseline allows IT administrators to document the initial system state, making troubleshooting, auditing, and future comparisons easier.

---

# Scenario

A new Windows 11 Enterprise workstation has been installed.

Before deploying the device to an employee, the IT department must document the system's hardware, operating system, network configuration, and security status.
# Skills Learned

- Windows Administration
- Command Prompt

- Hardware Inventory

- Enterprise Baseline Documentation

---

# Enterprise Best Practices

- Always document a workstation before making configuration changes.
- Record operating system version and build number.
- Verify Secure Boot status.
- Record hostname and logged-on user.
- Document network configuration.
- Keep baseline documentation for troubleshooting and auditing.
