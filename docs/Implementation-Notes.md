# Implementation Notes

## Overview

This document summarizes the implementation process of the Windows 11 IT Support Lab project.

## Environment

- Operating System: Windows 11 Pro
- Virtualization Platform: VirtualBox
- User Account: ITAdmin
- Network Configuration: DHCP and Static IP
- File System: NTFS

## Implemented Tasks

- Windows installation and initial configuration
- User and local account management
- Windows Update configuration
- Driver verification
- File and folder permissions (NTFS)
- Disk management
- Windows services administration
- Local security policy configuration
- Network troubleshooting
- Windows Defender Firewall configuration
- Event Viewer log analysis
- Performance Monitor analysis
- BitLocker verification

## Validation

Each task was tested and verified before proceeding to the next phase. Screenshots were captured as implementation evidence.
